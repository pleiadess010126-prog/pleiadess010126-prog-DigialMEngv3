**Q: Can I change which 3 platforms are connected?**
```

**Q: What happens if I hit my content limit?**
```
A: You'll see a warning when approaching the limit.

**Q: Can I edit AI-generated content before publishing?**
```

**Q: What's the difference between GEO and SEO scores?**
```
A: SEO = Optimized for Google Search ranking
   GEO = Optimized for AI assistant citations (ChatGPT, etc.)
   Both are important for modern content strategy.
```

**Q: Can team members see my billing information?**
```
A: No. Only Admins can see billing info.
   Editors and Viewers don't have billing access.
```

**Q: How do I cancel my subscription?**
```
A: Go to Settings > Usage & Billing > Manage Subscription > Cancel.
   You'll keep access until the end of your billing period.
